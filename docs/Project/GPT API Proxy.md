### Feature

实现自定义管理权限的可行方案
您的问题是关于如何让用户获得类似 root 权限来管理 channel ，而不需要为每个用户部署单独的数据库环境。子管理员角色扩展, user_root 角色, root 可以为授权的用户开启托管模式, 授权后可展示 channels 页面, 并获取配置修改权限, 数据库需要创建新表 user_channels , 以用户 id, ... 等信关联号用户,  其它数据和 channels 相同
